## Cause
Occurs when system RAM is heavily utilised by running applications or background processes.

## Resolution Steps
1. Close unnecessary applications using Task Manager
2. Restart the device
3. Increase virtual memory manually
4. Disable unnecessary startup applications
5. Install Windows and driver updates
6. Run a full malware scan
7. Run Disk Cleanup
8. Check for memory leaks
9. Consider a physical RAM upgrade if supported

## Notes
The warning should no longer appear once memory pressure is removed.
