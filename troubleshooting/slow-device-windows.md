## Symptoms
- Device running slowly
- Delayed application launch
- High CPU or disk usage
- Freezing or crashing

## Troubleshooting Steps
1. Restart the device
2. Install all Windows updates and driver updates
3. Perform a full antivirus scan
4. Disable unnecessary startup applications using Task Manager
5. Run Disk Cleanup to remove temporary files
6. Reduce visual effects for performance
7. Increase virtual memory (paging file)
8. Run Windows System Maintenance
9. Check disk health using CHKDSK
10. Perform a clean boot to isolate software conflicts
