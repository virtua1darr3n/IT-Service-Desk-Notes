# Proxy Server Pop-Up Prompts

## Easy Fix
1. Open Credential Manager
2. Select Windows Credentials
3. Remove cached credentials (do not remove certificate-based credentials)
4. Restart the device

## Advanced Fix
1. Disable proxy under Windows Settings
2. Reset Internet Options LAN settings
3. Scan for malware
4. Restart and retest

## Notes
This issue is commonly linked to cached credentials or misconfigured proxy policies.
